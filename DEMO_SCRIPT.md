# Digital Arena — Demo Script
### Пошаговый сценарий для инвесторской презентации

---

## Перед демо
1. Убедитесь, что бот запущен (Render dashboard → зеленый статус).
2. Откройте Telegram. Установите приложение (Add to Home Screen) на телефоне.
3. Запустите `seed_zone_photos.py` чтобы добавить красивые фотографии зон:
   ```
   python seed_zone_photos.py
   ```

---

## Сценарий (порядок действий)

### Шаг 1: Регистрация нового пользователя
1. Открыть Telegram → найти бота.
2. Нажать `/start`.
3. **Показать:** Приветственное сообщение с кнопкой возрастной верификации.
4. Нажать ✅ "Мне есть 18 лет". 
5. Бот запрашивает номер телефона → нажать кнопку "Поделиться номером".  
   **Говорим:** *"Пользователь верифицирован — мы знаем его реальный номер. Никаких фейков."*

### Шаг 2: Выбор клуба и зоны с фото
1. Нажать "🏢 Клубы".
2. Выбрать клуб из списка.
3. Нажать "Смотреть места".
4. **Показать:** Список зон (VIP / Standard / Bootcamp).
5. Нажать на любую зону (например, VIP).  
   **Показать:** Красивое фото зоны + описание + список ПК.  
   *"Владелец клуба загружает фото — пользователь сразу видит атмосферу."*

### Шаг 3: Бронирование в боте
1. Выбрать ПК (например, VIP-1).
2. Выбрать сегодня.
3. Выбрать время (например, 15:00).
4. Выбрать длительность (1 час).
5. **Показать:** Подтверждение брони. Появляется кнопка "🎟 Показать код".
6. Нажать "Показать код" → **QR-код** с кодом подтверждения.  
   *"Клиент приходит в клуб, показывает QR — администратор сканирует. Никакой путаницы."*

### Шаг 4: Бронирование через веб-приложение (PWA)
1. Открыть ссылку `https://digital-arena-njok.onrender.com` в браузере телефона.
2. **Показать:** Чистый интерфейс PWA.
3. Или открыть Mini App прямо через Telegram — кнопка в меню.
4. В Telegram нажать "📱 Приложение" → бот отправляет 6-значный код.  
5. Ввести номер и код на сайте → вход!  
   *"Работает в Telegram, на Android, iOS, Windows — без скачивания из App Store."*

### Шаг 5: Статистика для администратора
1. В Telegram отправить `/admin`.
2. Нажать "📊 Статистика".
3. **Показать:**
   - Кол-во пользователей, активных сеансов.
   - Брони сегодня/за неделю.
   - Выручка в сумах по каждому клубу.
   - Топ-3 популярных ПК.  
   *"Владелец клуба открывает бота и видит деньги прямо сейчас."*

### Шаг 6: Admin — управление клубом
1. Нажать "🏢 Клубы".
2. Выбрать клуб → "⚙️ Настройки".
3. **Показать:** Можно задать описание, часы работы, Wi-Fi, фото клуба.
4. Нажать "📦 Компьютеры" → список всех ПК с возможностью включить/выключить.

## Если что-то пойдет не так
- **Бот не отвечает:** Render может засыпать (free tier). Открыть `https://digital-arena-njok.onrender.com/api/health` — это разбудит сервис.
- **Фото не появляется:** Запустить `python seed_zone_photos.py`.
- **Ошибка миграции:** Запустить `alembic upgrade head` вручную.
