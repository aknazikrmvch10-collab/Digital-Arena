# Список файлов для переезда (Migration Checklist)

Для корректной работы проекта на новом месте необходимо перенести следующие файлы и папки. 

> [!IMPORTANT]
> **Не переносите** папку `venv` и `__pycache__`. На новом месте нужно будет создать новое виртуальное окружение и установить зависимости из `requirements.txt`.

## 📂 Папки (Обязательно)
- `handlers/` — основная логика бота и API.
- `drivers/` — интеграции с iCafe и др.
- `keyboards/` — меню Телеграм бота.
- `utils/` — вспомогательные функции.
- `website/` — файлы основного сайта.
- `miniapp/` — файлы Telegram Mini App.
- `firebase_public/` — файлы для деплоя на Firebase Hosting.
- `alembic/` — миграции базы данных.

## 📄 Файлы (Обязательно)
- `.env` — **КРИТИЧЕСКИ ВАЖНО**. Здесь хранятся ваши токены и доступ к БД.
- `main.py` — точка входа (Бот + FastAPI).
- `config.py` — настройки проекта.
- `models.py` — структура таблиц базы данных.
- `database.py` — подключение к БД.
- `requirements.txt` — список библиотек для установки (`pip install -r requirements.txt`).
- `firebase.json` & `.firebaserc` — конфигурация Firebase.
- `alembic.ini` — настройки миграций базы данных.
