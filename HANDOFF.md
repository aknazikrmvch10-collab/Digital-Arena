## 🛠 Как тебе (или твоему ИИ-агенту) синхронизировать проект

Чтобы твой локальный проект заработал с нашими новыми фичами, выполни следующие шаги по порядку:

### Шаг 1: Получить свежий код
Убедись, что твой друг сделал `git push` своих изменений.
В твоем терминале (Git Bash или терминал редактора) выполни:
```bash
git pull origin main
```

### Шаг 2: Обновить зависимости (Важно!)
Мы добавили библиотеку `qrcode[pil]` для генерации картинок и обновили версию `slowapi`.
Выполни это в терминале, находясь в папке с проектом:
```bash
pip install -r requirements.txt
```

### Шаг 3: Обновить базу данных (Миграция)
Мы добавили новую колонку `confirmation_code` в таблицу `bookings`. Тебе нужно создать и применить миграцию локально:
```bash
alembic revision --autogenerate -m "add confirmation_code"
alembic upgrade head
```
*(Если `alembic` не работает напрямую, используй `python -m alembic ...` или убедись, что активировано виртуальное окружение `.\venv\Scripts\activate`)*

### Шаг 4: Настройка `.env`
Скопируй содержимое `.env.example` в свой локальный файл `.env` и подставь свой тестовый `BOT_TOKEN` и URL локальной базы данных.
